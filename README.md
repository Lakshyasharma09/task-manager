# Team Task Manager

## Overview
This is a full-stack Team Task Management web application where users can create projects, assign tasks, and track progress.

## Features
- User Signup & Login (JWT Authentication)
- Role-based access (Admin)
- Project creation and management
- Task assignment to users
- Task status update (TODO, IN_PROGRESS, DONE)
- Dashboard with task statistics

## Tech Stack
- Backend: Spring Boot
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Authentication: JWT

## How to Run

### Backend
- Run Spring Boot application
- Configure database in application.properties

### Frontend
- Open index.html in browser

## API Endpoints
- POST /auth/signup
- POST /auth/login
- GET /projects
- POST /projects
- GET /tasks
- POST /tasks
- GET /dashboard
